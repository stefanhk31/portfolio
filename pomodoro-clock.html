Pomodoro Clock

Based on the Pomodoro Technique, this timer app allows users to set times (up to one hour) for work sessions and breaks. When a user clicks the start button, the session timer counts down. When the timer hits zero, it switches to break mode, and switches back once the break timer is complete.

I built this timer using React, with Bootstrap elements added for the buttons. The project took me about three weeks. It was an excellent opportunity to learn more about passing props into child components in React, as well to understand the cons of JavaScript's Interval function (I used Moment.js to ensure an accurate countdown).
